# neurofoundations_perspective

This is the code accompanying the paper *Neuro-foundation models as a critical step toward building generalizable models of the neural code* (Bashivan et al., Nature Neuroscience, in press).

## Contents

- [`costs_analysis/`](costs_analysis/) — closed-form cost model for four neuro-foundation-model data strategies (reproduces Box 7). See its README for setup, demo, and expected run time.
- [`diversity_analysis/`](diversity_analysis/) — PubMed + LLM pipeline quantifying paradigm concentration across neuroscience subfields (reproduces Box 9 / Fig. S1). See its README for setup, a no-GPU quick demo, and expected run time.
